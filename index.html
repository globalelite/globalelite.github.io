中止だ中止

WORK 鯨の仔

ジャンル
同人SF小説アンソロジー
紹介
終焉を迎える人類。その世界の歪な風習と新たな世界の夜明けの接地点を描く表題作を始め、今回もアクの強い作品が集結。文明の豊かさと負債の狭間で苦しむ青年とロボの友情に、ウザい老人の奇想天外な末路。ＶＲゲームでバグ技を追求する変人たちの軌跡もあれば、業務規則に過適合しトラブルを粉砕する宇宙運送屋もいる。異常な世界観を作って良いのは、現実世界ではなく最強同人誌の中だけにしろ。そう強く願う。
収録作品
  1. GOOD TIME（東京ニトロ）
  2. マンガで分かる「植物のふしぎ」（元壱路）
  3. ユニテイル・トーナメント五周年記念特集（髙𫝶創）
  4. 恋は流れ星のように（架旗透）
  5. 鯨の仔（維嶋津）
刊行日
二〇二二年五月二十九日

WORK フォーワン

ジャンル
同人SF小説アンソロジー
紹介
モデルロケットでスマートフォンを宙へ送ろう。理系科目が得意な少年たちによる、中学から高校にかけて行われた挑戦を描いた表題作をはじめ、今回も収録作はアクセル全開。魔法世界の妖精が巻き込まれる事件、サーカスで活躍する人工生物の本能に、近未来の展示会に生じる謎の幻影。そして、インフルエンサーと作り上げる艶かしい香りまで。定期シリーズＷＯＲＫに初の寄稿者を迎え、俺たちは“最強”へ突き進む。

WORK マンモス大合成

ジャンル
同人SF小説アンソロジー
紹介
人間とは何か。それはマンモスであり、ネコであり、記憶であり、流浪者であり、行動であり、そしてなにより、どいつもこいつも病人である。逃げろ。破壊される街から。狂人だらけの病院から。失敗の幻影から。追ってくる誰かから。八方ふさがりの運命から。間違ったままに進む先にも未来はある。良いか悪いかは別として。正義と不実と真実と虚無をまぜこぜに、時代のその先を圧倒的な『パワー』で描く五篇を収録。

TASK 令和時代

ジャンル
同人小説テーマアンソロジー
紹介
ＳＦサークルグローバルエリート合同誌第五弾は初となる「テーマ短編集」。令和時代という言葉をもとにサークル内外から十三篇の作品を収録。ＳＦだけでなく、ホラーやミステリ、日常モノにファンタジーと、様々なジャンルから新時代の「これまで」と「これから」を描く。そこにあるのは、ディストピアかユートピアか、あるいは変わらぬ日常か……。奔放な想像力が生む、色とりどりのイメージに刮目せよ！

WORK Upside Down

ジャンル
同人SF小説アンソロジー

猫。ジャムパン。姉妹。完璧な球。そして、僕。ロンドンの美大生が出会った、春先の奇妙な罠。表題作はじめ、奇妙な七つの「未来」と「世界」を収録。断末魔コンテストに憧れる青年、複数の体をデバイスとして持つ「家族」たち、AIに人権があると主張する弁護士に、全宇宙崩壊の危機に立ち向かう電子人格。そしてテラフォーミングされる安アパート……。歪になった世界で、はたして「帰還者＝あなた」は何を思う？

WORK 堕ち人賛歌

ジャンル
同人SF小説アンソロジー
紹介
超光速宇宙船が実用化された未来。中国政府が主導する太陽系外惑星調査団の一行は、予想外のファーストコンタクトに直面する。ＳＦサークルグローバルエリート合同誌第三弾は、新メンバーを加えさらにパワーアップ。バイオハックで妹を改造しようとする青年、宇宙人侵略のなか交錯する画家と研究者たちの人生、包囲された限界集落から脱出しようとする兄弟たち、自動運転社会で騒動に巻き込まれる会社員……未来社会の人生模様を切り取った五篇を収録。
